ExamEase – Smart Study Planner

ExamEase is a browser-based study planner built with HTML, CSS, and JavaScript. It helps students organize exam preparation by creating study schedules, setting reminders, tracking progress, and receiving adaptive time suggestions. All data is saved locally using the browser's storage.

Overview

ExamEase allows users to:

Add and manage multiple subjects

Create study sessions with date, time, and duration

Set reminders for upcoming study tasks

Track progress through visual indicators

View weekly analytics

Export and import data using JSON

Sync sessions to Google Calendar

Receive adaptive suggestions based on exam dates and syllabus load

How to Use

Open the file examease.html in any modern browser.

Add subjects and schedule study sessions.

Allow notifications for reminders.

Use the dashboard, calendar, and analytics to monitor progress.

Use export/import options to save or restore your data.

Limitations

Reminders only work while the page is open.

No cloud sync or user accounts.

No automatic detection of overlapping sessions.

Conclusion
